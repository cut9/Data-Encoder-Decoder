# Data-Encoder-Decoder (Windows Forms)

Утилита для кодирования/декодирования данных (примеры: base64, простые шифры, кодировки).

[![.NET CI](https://github.com/cut9/Data-Encoder-Decoder/actions/workflows/dotnet-ci-windows.yml/badge.svg)](https://github.com/cut9/Data-Encoder-Decoder/actions)
[![Release](https://img.shields.io/github/v/release/cut9/Data-Encoder-Decoder)](https://github.com/cut9/Data-Encoder-Decoder/releases)

---

## Информация
- Платформа: Windows (WinForms)
- Язык: C# .NET 8
- Тип: GUI-приложение (окно Windows Forms)

---

## Быстрый старт — запустить релиз (рекомендуется для обычных пользователей)

1. Перейдите в раздел **Releases**: https://github.com/cut9/Data-Encoder-Decoder/releases  
2. Скачайте архив `win-x64.rar` для вашей платформы.  
3. Распакуйте архив и дважды кликните `Developer Toolbox.exe`.

---

## Быстрый старт — запустить из исходников (для разработчиков)

### Команды (PowerShell / CMD)
```powershell
git clone https://github.com/cut9/Data-Encoder-Decoder.git
cd Data-Encoder-Decoder
dotnet restore
dotnet run --project ./"Developer Toolbox"/"Developer Toolbox.csproj"
```
